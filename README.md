# Hackaton #1

За каждым из разработчиков привязаны ветки разработки.
Каждый разработчик ведет разработку своей фитчи строго на своей ветки.
`git clone` - Команда, чтобы скопировать репозиторий к себе на локальную машину
`git checkout имя ветки` - переключиться на ветку

<img width="1385" height="638" alt="image" src="https://github.com/user-attachments/assets/a59c5bd6-fce1-4ed1-a98b-0efe40165102" />

После того, как фитча будет считаться завершенной, разработчк должен отправить `pull request` на ветку `development`

1) Ветка `call_sound` - Разработчк: Лидия, задача: Реализовать в папке modules новый модульный файл, который будет отвечать за вызов рандомного звука при клике на компонент модуля

2) Ветка `context_menu` - Разработчик: Степан, задача: Реализовать функционал контекстного меню, разработать для себя тестовый модуль, который нужно будет вставить в menu и проверить, что всё работает. Реализация осуществляестья в файле `menu.js`

3) Ветка `call_message` - Разработчик: Маргарита, задача: Реализация модуля в папке `modules` создать файл, который будет отвечать за модуль, при нажатии на который, пользователю будет выводиться рандомное сообщение из придуманного вами списка сообщений.
   
4) Ветка `change_color` - Разработчик: Мария, задача: Реализация модуля в файле `background.module.js`, при клике на который происходит изменение цвета страницы на рандомный цвет.
   
   
5) Ветка `create_clicks` - Разработчки: Иван, задача: Реализация модуля, при клике на который запускаеться таймер в 3 секунду и начинаеться счёт кликов пользователя, по истечению времени пользовователю выводиться статистика: Сколько времени прошло и сколько он сделал кликов 

6) Ветка `create_shape` - Разработчик: Андрей, задача: Реализация модуля, при клике на который в случайном месте экрана создаеться случайная фигура, случайного размера и случайного цвета.

**Создать PullRequest на ветку development до 29.11.2025 22:00**
