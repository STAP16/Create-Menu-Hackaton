# Hackaton #1

За каждым из разработчиков привязаны ветки разработки.
Каждый разработчик ведет разработку своей фитчи строго на своей ветки.
`git clone` - Команда, чтобы скопировать репозиторий к себе на локальную машину
`git checkout имя ветки` - переключиться на ветку

Данная ветка создана от базовой версии Хакатона!
На данной ветки ведуться разработки дополнений к основному приложению.

_По всем вопросам в телеграм чат хакатона_
Счастливой разработки!
